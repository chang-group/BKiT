Protein-ligand dissociation thermodynamics and kinetics analysis toolkit

## Dependancies
- `python` 3.6 >
  - `pytraj` 2.0.5

## Installation

Clone the repository:

*git clone https://github.com/truzmeto/BKit.git*

then run the following command:

*python setup.py install*
